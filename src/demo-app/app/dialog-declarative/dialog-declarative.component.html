Attributes of the mdl-dialog component
Attribute Description
mdl-modal Set the dialog to modal or not. Default value is modal.
hide Event will be triggered if the dialog is hidden.
show Event will be triggered if the dialog is visible.
dialog-width Set a custom width to this modal. Default minimum width is preset from material-design-lite
